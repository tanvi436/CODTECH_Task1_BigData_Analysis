# CODTECH Task 1 - Big Data Analysis using PySpark

## Project Title
Big Data Analysis Task using PySpark (Taxi Dataset)

## Description
This project performs cleaning and analysis of a Taxi dataset using PySpark.
It includes data preprocessing, handling missing values, and saving the cleaned dataset
into Parquet and CSV format.

## Technologies Used
- Python
- PySpark
- Hadoop (winutils)
- Jupyter Notebook

## Output Files
- cleaned_taxi_data.parquet
- cleaned_taxi_data_final.csv

## How to Run
1. Install PySpark
2. Configure Hadoop path (winutils)
3. Run the Jupyter Notebook file: `CODTECH_Task1_BigData_Analysis.ipynb`
